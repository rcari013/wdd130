How it works!

1. Review

This is the process where I need to check how the opeation goes. This includes checking the involved tools, websites, processes, employees and many other factors.

2. Evaluation

This is to provide you feedback on your business process by using a systematic approach to check what can be improved.

3. Research

This is where I need to see other alternatives in the market that might still do the job but can save more on cost. This will incur a study on how some process can be improved. We compiled things we researched and learned.

4. Proposal

Once studies and research are done, this is where we will propose changes to you. If you agree then the next step is to implement Implementation - This is where we implement the changes

5. Monitor

Where we track and log changes. We will compare the changes from the previous reports and data to the present. Repeat the whole step You can click the button below for further discussion.
